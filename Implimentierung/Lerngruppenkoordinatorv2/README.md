# Lerngruppen-Koordinator
## Jonas Becker
## Beschreibung 

Im Lerngruppenkoordinator haben Schüler/Schülerinnen die Möglichkeit Lerngruppen zu verwalten.
Dabei kann sich der Benutzer anmelden, seine eigenen Lerngruppen erstellen, Lerngruppen beitreten und verlassen.
Zusätzlich können PDFs als Unterrichtsmaterial hochgeladen und geöffnet werden.
![Screenshot Benutzer](../../Dokumentation/Base.JPG)

## Nutzung
### 1. Benutzer Anmelden

Beim Starten des Lerngruppenkoordinators wird das System auf die nötige Datenbank geprüft. Danach werden alle verfügbaren Lerngruppen
geladen und angezeigt. Bevor der Nutzer aber eine Gruppe erstellen oder einer Gruppe beitreten kann, muss er sich anmelden. 
Das Anmelden erfolgt über den Knopf "Nutzer ändern" und zeigt ein neues Fenster auf.

![Screenshot Anmeldeknopf](../../Dokumentation/Anmeldeknopf.JPG)

Dort können Nutzer geladen (2) oder neu erstellt werden (1):

![Screenshot Optionen](../../Dokumentation/OptionenAnmelden.JPG)

### 2. Lerngruppe beitreten

Zum Beitreten einer Lerngruppe benötigt es nur einen Klick auf den Haken in der Lerngruppe, der man beitreten möchte:

![Screenshot Beitreten](../../Dokumentation/BeitretenKnopf.JPG)

Die ausgewählte Lerngruppe wird unter "Meine Lerngruppen" gelistet:

![Screenshot Meine Gruppen](../../Dokumentation/MeineGruppen.JPG)

Zum Austreten einer Lerngruppe kann der Haken bei "Verlassen" gedrückt werden.

### 3. Lerngruppe erstellen

Um eine Gruppe zu erstellen drückt man den "Gruppe Erstellen" Knopf unter den beigetretenen Gruppen:
![Screenshot Gruppe Erstellen](../../Dokumentation/GruppeErstellen.JPG)

Im Gruppen-Erstellen-Fenster kann man die Daten für die neue Lerngruppe eintragen und speichern

![Screenshot Gruppe Erstellen](../../Dokumentation/GruppenErstellenMenu.JPG)
### 4. Gruppe suchen

Um explizit Gruppen zu suchen kann man über "Lerngruppen Suchen" direkt ins Suchfenster:
![Screenshot Suchen](../../Dokumentation/LerngruppeSuchen.JPG)

### 5. PDF Funktion

Um eine PDF von einer Lerngruppe runterzuladen kann auf den Downloadknopf in der jeweiligen Lerngruppe gedrückt werden:

![Screenshot Meine Gruppen](../../Dokumentation/PDF-Anzeigen.JPG)
